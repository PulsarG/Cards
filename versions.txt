ver 08 - 
            Оптимизирована выгрузка БД в структуру.
            Теперь это происходит в отдельной функции LoadWord
            Создается сразу два слайса слов: полный (для страницы со списком), и с исключением слов с частотой 0 (для главной страницы)
            Функция выбора рандомного слова NextWord() из массива слов, без слов с частотой 0, так же 
               добавлена сразу в функцию LoadWord.
            Теперь LoadWord возвращает три значения - полный массив слов БД, рандомное слово из массива слов исключающих 0 частоту, и err.
            Таким образом функции страниц MainPage и List получают уже готовые данные
               для вывода и не выполняют функции загрузок и выбора заного каждый раз.
